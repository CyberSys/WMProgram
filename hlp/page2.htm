Настройка мониторов



Рис. 2. Настройка мониторов

Предварительно, для всех моделей видеокарт, рекомендуется скачать и установить последнию версию драйверов с сайта производителя. При установке надо выбрать ручной выбор устанавливаемых компонентов и оставить только драйвера, от установки различных панелей управления, предлагаемых производителем необходимо отказаться.

Данная вкладка позволяет определить для каждого из мониторов рабочее место, на котором он будет использоваться. В верхней части окна находится список всех подключенных к компьютеру мониторов. Название монитора состоит из названия видеокарты и названия монитора.

Для того, чтобы указать какому рабочему месту принадлежит монитор, надо выбрать этот монитор в верхнем списке, кликнув по нему левой кнопкой мыши, и выбрать рабочее место из выпадающего списка, находящегося под списком мониторов.

При использовании видеокарт ATI с двумя видеовыходами могут возникнуть проблемы с отображением курсоров на рабочих местах, использующих одну видеокарту. Для решения этой проблемы служит опция "Отключить аппаратный курсор для устройства". Сначала необходимо включить её для одного из мониторов, если после этого проблемы не исчезнут, надо попробовать отключить курсор и на другом мониторе. Это можно сделать с помощью той же опции или нажав кнопку "Отключить аппаратный курсор на всех устройствах", которая отключит аппаратный курсор в настройках ускорения для всех мониторов.

Внимание! Так же, при использовании некоторых видеокарт семейства ATI (X700 Series ...) первому(основному) рабочему месту надо назначать обязательно первый монитор, подключенный к этой видеокарте (тот, в названии которого нету слова Secondary), иначе возможны проблемы с изображением при запуске системы.

После назначения для мониторов рабочих мест, на которых они используются, нужно сделать то же самое для устройст ввода на следующей вкладке.
