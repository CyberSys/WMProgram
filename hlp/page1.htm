Основные настройки

Рис. 1. Основные настройки

Вкладка позволяет указать основные настройки программы и запустить вручную дополнительные рабочие места.

Внимание! Перед настройкой необходимо убедиться, что все устройства, необходимые для дополнительных рабочих мест, подключены и нормально функционируют. Во время настройки не желательно подключать и устанавливать какие-либо устройства. Если же это необходимо сделать, то надо выйти из программы настройки.

Количество рабочих мест - определяет количество мест, включая и основное место. Всего можно создавать до 9 дополнительных мест. Эта опция должна быть установлена самой первой после запуска программы.

Показать номера рабочих мест - при нажатии на данную кнопку на мониторе каждой из станций появляется сообщение с номером рабочего места. Данная кнопка активна только при запущенных дополнительных рабочих местах.

Включить/Выключить - с помощью этой кнопки осуществляется управление сервисом. Запускать дополнительные рабочие места можно только при включенном состоянии сервиса.

Запускать при старте системы - При установке этой опции дополнительные рабочие места будут запускаться при старте системы. Данная опция активна только при включенном состоянии сервиса.

Запустить сейчас - запустить дополнительные рабочие места вручную. Кнопка активна только при включенном состоянии сервиса.

После установки всех вышеобозначенных опций необходимо перейти к следующей вкладке - "Настройка мониторов". Где для каждого монитора необходимо указать рабочее место, на котором он будет использоваться.
