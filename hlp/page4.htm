Рис. 4. Настройка IP адресов

Иногда возникает необходимость назначить каждому из рабочих мест отдельный IP адрес. Например, это может понадобиться для раздельного учёта интернет траффика. Программа позволяет назначить IP адрес каждому из рабочих мест.

Перед назначением IP адресов рабочим местам необходимо добавить дополнительные IP адреса для сетевого подключения. Для этого нужно на вкладке "Общие" свойств сетевого подключения выбрать пункт Протокол интернета (TCP\IP) -> Свойства -> Дополнительно -> Параметры IP -> Добавить..

Опция "Использовать раздельные IP адреса" включает разделение IP адресов по разным рабочим местам. После включения опции надо выбирать рабочие места из списка и указывать для каждого из них IP адрес.

Внимание! Использование раздельных IP адресов возможно только для одной сетевой карты.
